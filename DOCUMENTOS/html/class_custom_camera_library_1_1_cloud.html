Detailed Description

Clase que representa una nube de puntos sacada con el pointer. Cuenta con propiedades que la definen y permite ir representando cada nube que se vaya a sacar con la ayuda del pointer.

◆ savePoint()

bool Cloud::savePoint ( Point3d  point,
int  key 
)

Guarda un punto dentro de la nube de puntos.

Parameters
pointel punto que se intentará guardar.
keyel valor ascii de la tecla que se presione, -1 si no se oprime nada. Este valor es utilizado para controlar el comportamiento de la toma de puntos (pausar, reanudar y guardar). Los teclas utilizadas son:
  • x: Captura un punto en modo manual (se escucha un sonido por cada punto tomado con éxito).
  • z: Pausa/reanuda la toma de puntos.
  • a: Guarda los punto que han sido tomados hasta el momento.
Returns
verdadero si el punto fue guardado o falso si sucede lo contrario.
