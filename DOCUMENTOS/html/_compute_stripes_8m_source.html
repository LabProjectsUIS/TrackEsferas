4 %
5 % This function computes the projector stripe coordinate (at subpixel
6 % accuracy) at every pixel in the image. The algorithm used in temporal
7 % processing (with a translationnal pattern of period 32 pixels). A coarse
8 % to fine pattern projection helpresolve for the period ambiguity (in a
9 % Gray-Code Scheme).
10 %
11 % The naming convention is,
12 %
13 % fname_pat##p.ras --> for the positive image from pass ##
14 % fname_pat##n.ras --> for the negative image from pass ##
15 %
16 % ##=00: Black and White images
17 %
18 % ##=[01 - 10] : Coarse to fine projection (Gary-Scale projection)
19 % ##=[11 - 42 ]: 32 translational patterns of period 32 pixels (for
20 % temporal processing)
21 % INPUT:
22 % fname --> base name of the images
23 % graphout --> Set to 1 to show graphical figures

25 % OUTPUT:
26 % xc is a 2 by N matrix of the point in the image plane
27 % (convention: (0,0) -> center of the upper left pixel in the camera image)
28 % xp is a N vector of the corresponding projector stripe numbers (at subpixel accuracy).
